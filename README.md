# Superstore Sales & Profit Analysis 📊

## 📌 Overview
This project analyzes retail sales data using Python, Pandas, and Plotly.

## 🔍 Features
- Data Cleaning & Preprocessing
- Sales Trend Analysis
- Profit Analysis
- Data Visualization

## 🛠 Tools Used
- Python
- Pandas
- Plotly

## 📊 Key Insights
- Technology category generates highest revenue
- Furniture has low profit margins
- Consumer segment contributes most sales

## 📁 Files Included
- Project Notebook (.ipynb)
- Dataset (.csv)
- Report (.pdf/.docx)
